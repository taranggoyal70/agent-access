# 60-second YC demo

## 0–8 seconds: problem

“SaaS companies give external agents broad API keys or make them impersonate users. They cannot prove who authorized an action or safely expose only part of the product.”

## 8–22 seconds: import

Open the Import page. Paste the Design Partner's real OpenAPI URL and run preflight. Point to the detected staging origin, authentication scheme, read/write risk counts, and blocked changes.

## 22–34 seconds: connect

Select the staging credential type, connect, and show the active Vendor Connection. Explain that only the encrypted server-side credential reaches staging and Shadow Mode disables writes.

## 34–50 seconds: external agent

Open the published discovery manifest from a separate agent client. Register the agent, receive a one-hour delegated credential, and invoke one real read-only capability.

## 50–60 seconds: proof

Show the signed receipt: Principal, Agent Account, Delegation, capability, upstream status, latency, body hashes, and policy decision. End with the weekly verified upstream execution metric.

Do not use the synthetic sample in the YC demo. Use a named Design Partner with permission.
